# The theory behind these “7.5 Andalusian cadences”

## 1) The one sentence theory

All of these progressions are the **Andalusian cadence**: a **descending tetrachord** harmony that expresses **Phrygian tonic gravity**.

In practice, the skeleton is:

**Phrygian cadence (relative to tonic):**

```
iv  →  ♭III  →  ♭II  →  I(♭2 flavor)
```

Or in pitch terms (if tonic = E):

```
Am  →  G  →  F  →  E
```

That’s the “Andalusian cadence” everybody means.

---

## 2) Why it feels like it “resolves” even though it’s not V→I

This cadence is not classical dominant function. It’s **modal function**.

### The real “resolution engine”

The strongest pull is:

* **♭II → I** (half-step above tonic resolving down)

That is the defining sound of **Phrygian** (and by extension flamenco).

So instead of:

* V7 → I (leading tone up to tonic)

You get:

* ♭II (or ♭II color) → I (upper neighbor down to tonic)

That’s why the tonic chord is often **colored with ♭2/♭9**: it keeps the Phrygian bite present even at “home.”

---

## 3) Where the chords come from (mode source)

The Andalusian cadence is most naturally explained as coming from the **Phrygian mode**, or more specifically the flamenco tradition’s **Phrygian dominant / altered Phrygian colors**.

### E Phrygian (diatonic source)

E Phrygian scale:

```
E  F  G  A  B  C  D  E
```

Diatonic triads include:

* i = Em
* ♭II = F
* ♭III = G
* iv = Am

So:

```
Am (iv) → G (♭III) → F (♭II) → E (I as “tonic center”)
```

But flamenco often treats **E major** (E–G#–B) as the tonic sonority even though Phrygian is “minor-ish.” That hybrid is why it’s sometimes described as **“Phrygian with major tonic”**.

---

## 4) The “major tonic” issue (why the tonic is E major, A major, etc.)

In the transcriptions you provided, the tonic chord is often **major** with a **♭2/♭9** added:

* E(add ♭2)
* A(add ♭2)
* C#(add ♭2)
* etc.

### Why this works

Because flamenco tonality is not “major/minor functional harmony.” It’s a **tonic sonority + characteristic tension** system.

Think of it like:

* **Tonic chord = identity**
* **♭2 = signature color**
* **Resolution = upper neighbor down (♭2 → 1)**

So the tonic frequently includes the ♭2 to keep the mode “alive.”

---

## 5) What each chord is doing in the chain (function-by-function)

Using the generic model `iv → ♭III → ♭II → I`:

### iv (minor)

* sets the “dark” modal environment
* introduces the minor 3rd color that feels flamenco-ready

### ♭III (major)

* acts like a **brightness pivot**
* often shares tones with iv and ♭II, so voice-leading stays smooth

### ♭II (major or color dominant)

* **the real cadential driver**
* creates the semitone pull down to tonic

### I (major with ♭2 color)

* tonic, but not “clean”
* the cadence wants the “spice” to remain present

---

## 6) Why the video’s chord spellings add 7ths/9ths/#11

Those extensions aren’t random “jazzy” choices—they’re **voice-leading and color management**.

### Examples from your extracted set

#### C9 in the A/por medio set

C9 gives:

* a stable upper color tone
* smoother stepwise connection into the next chord
* makes the progression less “blocky”

#### Fmaj7(#11) (“F flamenco”)

That #11 (B natural over F) gives:

* a bright shimmer that avoids sounding like plain F major
* a characteristic “open” flamenco color (especially with open strings)

#### G13 in Taranta

G13 signals:

* dominant-color sonority without classical resolution demands
* keeps the chord useful for color and motion

In short: extensions are often there to support **guitaristic resonance** and **stepwise inner voices**.

## 7) The “7.5” explanation (why there’s a half)

In practice, the “half” usually means:

* a very popular **variation of one cadence**
* often differing by:

  * the quality of ♭II (maj vs maj7#11 vs 7alt)
  * or the tonic voicing (sus/♭9 vs plain)

So the theory is: **same skeleton, one common reharm**.

## 8) How to generalize this instantly (transposition rule)

If your tonic is **X**, then:

* iv = chord built on **X + 5 semitones** (a fourth up)
* ♭III = chord built on **X + 3 semitones**
* ♭II = chord built on **X + 1 semitone**
* I = chord built on **X**

So the cadence is a **bass descent**:

```
X+5 → X+3 → X+1 → X
```

That’s why it feels like a “slide down the neck.”

The “por medio / por arriba / por granaina / por taranta …” naming is primarily:

* **guitar-centric**
* tied to **tonic center + common voicings + where the bass sits**
* sometimes also tied to **traditional palo** (style) associations

Theory translation:

* the “por” names are a *practical tuning/position taxonomy*
* the academic name is: **Andalusian cadence / Phrygian cadence**

## Scale-degree bass motion core

### Phrygian tonic model (tonic = **1**)

Treat the tonic center as **1** (even if the chord is voiced major). Then the Andalusian cadence bass roots are:

**Bass degrees:**

```
iv   →  ♭III  →  ♭II  →  I
4    →   ♭3   →  ♭2   →  1
```

**Bass interval steps (in semitones):**

* 4 → ♭3 : down **2** semitones (whole step)
* ♭3 → ♭2 : down **2** semitones (whole step)
* ♭2 → 1 : down **1** semitone (half step)

So the bass motion is a **whole, whole, half** descent into tonic:

```
(whole) + (whole) + (half) downward
```

That’s the “Andalusian fingerprint.”
